# Le Bazar d'Infrep - Magasin Pédagogique

Bienvenue sur le site du **Bazar d'Infrep**, une boutique école dédiée à la formation professionnelle aux métiers du commerce, de la vente et de la mise en rayon.

## 🏪 Description

Ce projet est un site web moderne et responsif pour présenter notre espace de formation pédagogique. Il showcase les différents rayons du magasin et les articles disponibles dans notre boutique école.

## 🎯 Objectifs Pédagogiques

- Gestion complète d'un magasin (du balisage à l'implantation des produits)
- Maîtrise des techniques de mise en rayon
- Organisation et présentation de produits
- Apprentissage du commerce et de la vente

## 📋 Structure du Projet

```
Bazar-d-Infrep-/
├── index.html       # Page principale du site
└── README.md        # Ce fichier
```

## 🎨 Sections du Site

1. **Navigation** - Menu de navigation sticky avec liens vers les différentes sections
2. **Héro** - Bannière d'accueil avec présentation du projet
3. **Concept** - Explication de la boutique école
4. **Rayons** - Grille d'affichage des articles par rayon :
   - Prêt-à-porter (Textile)
   - Multimédia (Électronique)
   - Parfumerie (Beauté)
   - Hygiène (DPH)
   - Librairie (Culture)
   - Optique (Accessoires)
   - Bijouterie (Accessoires de Luxe)

## 🌐 Déploiement

### Via GitHub Pages

1. Accédez aux **Settings** de votre repository
2. Allez dans la section **Pages**
3. Sélectionnez `main` comme source
4. Votre site sera accessible à : `https://bryany712.github.io/Bazar-d-Infrep-/`

### Autres options
- Netlify
- Vercel
- Votre propre serveur web

## 🛠️ Personnalisation

### Modifier les couleurs
Modifiez les codes couleur directement dans la section `<style>` :
```css
header { background-color: #2c3e50; }
.logo { color: #e67e22; }
```

### Ajouter des produits
Ajoutez de nouvelles cartes produit dans la grille :
```html
<div class="product-card">
    <div class="product-img" style="background-color: #couleur;">Catégorie</div>
    <h3>Nom du Produit</h3>
    <span class="product-tag">Rayon</span>
</div>
```
